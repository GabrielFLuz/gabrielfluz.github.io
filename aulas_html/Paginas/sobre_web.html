Sobre a Web

A web é uma rede de informações interconectadas que permite o acesso a conteúdos diversos por meio de navegadores.
Ela revolucionou a forma como nos comunicamos, aprendemos e fazemos negócios, tornando-se uma parte essencial da vida moderna.
A WEB é quando acessamos os arquivos HTML feitos em outros computadores, e para isso a vários caminhos.
Primeiramente você precisa de um dispositivo, seja celular, computador, tablet ou notebook, com acesso à internet.
Após acessamos um browser, google chrome, edge, opera etc. Quando estamos no menu desse browser, isso já é uma página WEB,
e para termos acesso a essa página, seus arquivos ficam salvos em um servidor.

O servidor é um computador que armazena os arquivos e os disponibiliza para acesso na web. Quando você digita um endereço de site no navegador, o servidor envia os arquivos HTML
correspondentes para o seu dispositivo, permitindo que você visualize a página. A web é composta por milhões de sites, cada um com seu próprio conteúdo e propósito, desde blogs pessoais até grandes plataformas de comércio eletrônico.
Através da web, podemos acessar informações, interagir com outras pessoas, fazer compras, assistir a vídeos e muito mais.
