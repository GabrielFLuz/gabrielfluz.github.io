Sobre a WEB

A Web é um sistema global de documentos de hipertexto (páginas web) interligados por hiperlinks.
Esses documentos podem conter texto, imagens, vídeos, áudios e outros tipos de mídia, um exemplo
seria blogs, o Youtube entre outros, como blogs. A principal forma de acessar e interagir com a Web
é através de browsers como o Chrome, Firefox, Safari, Edge, Opera etc. Essas páginas são feitas
através de algumas como o HTML, CSS e JavaScript.
